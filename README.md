# 🖼️ ArtHub - Client Side

ArtHub is a premium digital platform designed to connect art lovers, collectors, and buyers with talented creators worldwide. The platform allows users to seamlessly browse, discover, and purchase original artworks. It empowers artists to showcase and manage their portfolios while providing admins with powerful tools to oversee the entire ecosystem.

---

## 🚀 Why ArtHub?

Traditional art buying is often restricted to physical galleries and limited exhibitions. ArtHub democratizes access to art by:
* **Global Reach:** Enabling emerging and established artists to reach a global audience.
* **Streamlined Experience:** Providing a secure, fast, and modern digital marketplace for buyers.
* **Advanced Features:** Implementing robust MERN/Next.js stack concepts, role-based access control (Admin, Artist, User), secure payment gateways, interactive comments, and live analytics.

---

## ✨ Key Features

### 👤 User / Buyer Features
* **Discover Art:** Browse and filter unique artworks by category, price, and artist.
* **Secure Purchase:** Integrated checkout system for a seamless buying experience.
* **Engagement:** Leave comments and feedback on favorite pieces.

### 🎨 Artist Dashboard
* **Artwork Management:** Dedicated portal to upload, update, and delete creations.
* **Personal Gallery:** Showcase unique artist profiles and specific collection links.
* **Live Updates:** Fast, dynamic updates without hard refreshes.

### 🛠️ Admin Panel
* **Ecosystem Control:** Full authority to manage users, roles, and review platform activities.
* **System Analytics:** Track overall sales, trending art, and platform engagement metrics.

---

## 🛠️ Tech Stack Used

* **Framework:** Next.js (App Router, Server & Client Components)
* **Styling:** Tailwind CSS (Modern, dark-themed responsive UI)
* **State Management & Routing:** Native Next.js Navigation and Context API / Hooks
* **Data Fetching:** Native Fetch API with custom route-handler logic (`no-store` dynamic rendering)
